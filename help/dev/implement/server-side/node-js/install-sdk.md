# 安装节点js。 SDK

Node.js SDK由以下方式分发 [npm](https://www.npmjs.com/package/@adobe/target-nodejs-sdk).

要开始使用此功能，请通过从NPM安装将其添加为依赖项：

## 安装

```js {line-numbers="true"}
npm i @adobe/target-nodejs-sdk -P
```

开源代码位于 <https://github.com/adobe/target-nodejs-sdk>.
